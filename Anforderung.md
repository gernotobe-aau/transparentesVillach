# Ziel des Systems

Eine statitische Website wo man Beiträge hinzufügen kann, eine Messagebox wo der User eine Message an den Betreiber bzw eine bestimme Email etwas schicken kann. Die Seite soll rechtlich vollkommen richtig sein mit Impressum, DSGVO,... 
Die Seite soll auch für mobile Geräte verwendbar sein. 
Es soll eine Art "Blog" Seite seien, wo ich Mitbürger aufkläre über Themen. 

## Projektname

Transparentes Villach

## Kurzbeschreibung

Es geht um Transparenz das Informationsfreiheitsgesetz, Anfragen an die Stadt Villach, Antworten der Stadt Villach,... Die Seite soll die Möglichkeit bieten, dass Mitbürger mir Anregungen, Informationen,.. zukommen bzw mir anonym eine Anfrage schicken und ich sie einreiche. IFG Anfragen können "anecken" bzw man will es sich mit der Stadt Villach nicht verscherzen, aber mir ist es egal. Mir ist Transparenz wichtig und mich stört es nicht, wenn die Stadt Villach mich nicht mag bzw ich als "Querulant" darstehe.

## Zielgruppe

Bürger der Stadt Villach bzw Umland, deswegen wäre Gelb/Gold als Farbe eine gute Wahl, weil die Stadt Villach es auch so hat.
Siehe www.villach.at

## Projektziele
- Eine einladende Seite
- Hochladen von Beiträgen bzw zum Hinzufügen.
- eine Messagebox wo der User den Betreiber eine Nachricht übermitteln kann (wahrscheinlich über Email bzw Formular) und es soll Anonym sein.
- eine rechtlich einwandfreie Seite, Impressum, Dsgvo,..
